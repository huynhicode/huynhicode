# Steph Huynh - UX Engineer

I work at the intersection of design and front-end engineering, usually thinking 
about both sides at once.

Most of my work lives in design systems and component libraries: building the 
foundations that make product UI consistent, accessible, and easier to maintain 
over time. I care about the details: clear component APIs, documented patterns, 
and accessibility baked in from the start.

**Current stack:** Svelte · Astro · TypeScript · Tailwind CSS · Storybook · Figma

## Focus areas

- Design systems and component libraries
- Accessible, responsive UI (WCAG)
- Design token architecture
- Documentation and implementation standards
- Bridging design intent and engineering execution
